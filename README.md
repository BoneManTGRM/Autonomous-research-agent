self-correcting discovery. It combines:

**1 hour → 8 hours → 24 hours → 1 week → 1 month → 90 days → 1 year**

Each run grows its own stability, reasoning depth, and long-run ΔR performance.

---

# 🔁 TGRM Loop — The Repair Core

Every autonomous cycle performs:

1. **TEST** — read current knowledge, detect weak points  
2. **DETECT** — find contradictions, gaps, logical instability  
3. **REPAIR** — gather evidence, run searches, ingest PDFs, compute biomarkers  
4. **VERIFY** — compute ΔR, E, and RYE to measure improvement  

Every cycle logs:

- ΔR — repair gain  
- E — cycle energy cost  
- **RYE = ΔR / E** — repair efficiency  
- Stability Index  
- Recovery Momentum  
- Oscillation Signature  

This creates a **closed-loop self-repairing intelligence system**, the heart of Reparodynamics.
